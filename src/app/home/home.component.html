The Volcano Petrology Database (VPD) facilitates storing and managing information about petrological data for ready accessibility to researchers. The data management system allows compiling multi-level dataset as images, analyses, results, and time series that can be interactively explored from the work space. The VPD offers three main features:

  • Users can organize data in work space from field work to analytical techniques by linking the files between them in an interactive work space. Users can create data plots for a better visualization.
  • Users can share the uploaded data with collaborators (choosing the level of privacy for each project).
  • Users can create an internal database with their collaborators.

The VPD offers two main menus allowing users to input, organize, plot, and query data.

  • Input: Users can create new projects for adding their data. When creating the project users can choose between different degrees of privacy/sharing. The listed volcanoes and eruptions are according to Smithsonian (https://volcano.si.edu), but users can add new volcanoes/eruptions. By using the work space and the associated tools users can link multi-level data and produce plots.
  • Access Database: Users can find the available data for a specific eruption or volcano. Users can query the data from the map or from one of the three lists: Volcano, Project, and Sample List.
*Users need to be registered to access the database and to upload data
